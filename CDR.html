Cat, Dog, and Raccoon Classifier

Source Code

This project is a fun Telegram bot that utilizes the YOLOv8 object detection model to classify images of cats, dogs, and raccoons. The bot allows users to send images via Telegram, which are then processed using YOLOv8 to detect and classify the animal in the picture. The system integrates Python, YOLOv8 for real-time object detection, and the Telegram API for interaction. This project demonstrates the application of cutting-edge object detection models for quick and accurate image classification in a user-friendly environment..
